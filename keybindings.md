# Keybindings for productivty 💻

## General Commands

### Close Tab
Cmd + w

### Open/Close Terminal
Cmd + Æ

### Toggle Terminal Fullscreen
Option + F

### Open sidebar
Cmd + B

## Copy and paste

### Copy file (in filetree)
Y + Y

### Paste file (in filetree)
P

### Delete file (in filetree)
D + D

### Change filename (in filetree)
C
